Emmet

Emmet es una potente herramienta que agiliza y optimiza el flujo de trabajo en el desarrollo web. Originalmente conocido como "Zen Coding", este conjunto de abreviaciones y atajos de teclado permite a los desarrolladores y diseñadores web escribir código HTML y CSS de manera más rápida y eficiente. Emmet es ampliamente utilizado en la comunidad de desarrollo web debido a su capacidad para aumentar la productividad y la facilidad de escritura de código. A continuación, se presenta una lista con un resumen de algunas de las principales abreviaciones de Emmet.

Markdown

Markdown es un lenguaje de marcado ligero utilizado para dar formato al texto en páginas web. Es fácil de usar y no requiere conocimientos avanzados de programación. Puedes usar Markdown para crear encabezados, listas, enlaces y otros elementos básicos de formato en tus documentos web.

Principales características de Markdown

  • Sintaxis sencilla: Markdown utiliza una sintaxis sencilla y legible que incluye caracteres especiales para aplicar diferentes estilos y formatos al texto.
  • Encabezados: Puedes crear encabezados de diferentes niveles utilizando uno o más símbolos de numeral (#). Cuantos más símbolos de numeral uses, más pequeño será el tamaño del encabezado, estos van desde el h1 hasta el h6 .
  • Listas: Markdown te permite crear listas ordenadas y no ordenadas de manera fácil. Las listas ordenadas se crean utilizando números seguidos de un punto (1., 2., 3., etc.), mientras que las listas no ordenadas se crean con guiones, asteriscos o signos de más (-, *, +).
  • Enlaces e imágenes: Insertar enlaces es sencillo: entre corchetes, escribes el texto del enlace y entre paréntesis, colocas la URL a la que debe dirigir. Para añadir imágenes, sigue una sintaxis similar, pero precede la URL con un signo de exclamación.
  • Énfasis y resaltado: Puedes enfatizar palabras o frases utilizando asteriscos (*) o guiones bajos (_). Un asterisco o guión bajo rodea una palabra o frase para aplicar énfasis, mientras que dos asteriscos o dos guiones bajos los resaltarán en negrita.
  • Bloques de código: Markdown permite incluir bloques de código en tu texto para resaltar sintaxis o mostrar ejemplos. Utiliza tres acentos graves (```) para abrir y cerrar bloques de código.
  • Separación de secciones: Puedes separar visualmente diferentes secciones de tu texto utilizando líneas horizontales. Usa tres guiones o tres asteriscos (--- o ***).
